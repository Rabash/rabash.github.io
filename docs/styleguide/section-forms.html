Section 6 forms Forms

Different parts of forms.

Section 6.1 forms.checkboxes Checkboxes

If an <input> element has the type='checkbox' attribute set, the form field is displayed as a checkbox.

Example
Weekdays

      <li class="c-form__item -checkbox">
        <input class="c-form__checkbox" id="monday" type="checkbox" name="monday" value="monday" checked="">
        <label class="c-form__label -option" for="monday">Monday</label>
      </li>
      <li class="c-form__item -checkbox">

Section 6.2 forms.description Description

Descriptions are sometimes needed as a complement to the label. Descriptions should make use of the aria-describedby-attribute to indicate a connection between input and description.

Example
Pick a username. 
<div class="c-form__item">
  <label class="c-form__label" for="text">Username</label>
  <input type="text" class="c-form__text" id="text" aria-labelledby="text-description" />
  <small class="c-form__description" id="text-description">Pick a username.</small>
</div>

Section 6.4 forms.fieldset Fieldsets

The <fieldset> element groups a set of form fields, optionally under a common name given by the <legend> element.

Example
A fieldset
Fieldset description
A disabled fieldset 
<fieldset class="c-fieldset ">
  <legend><span class="c-fieldset__legend">A fieldset</span></legend>
  <div class="c-fieldset__description">Fieldset description</div>
  <div class="c-form__item">
    <label class="c-form__label" for="fieldsetText">Text input</label>
    <input class="c-form__text" type="text" id="fieldsetText" placeholder="Text input">
  </div>
</fieldset>

<fieldset class="c-fieldset " disabled>
  <legend><span class="c-fieldset__legend">A disabled fieldset</span></legend>

Section 6.7 forms.labels Label

The <label> element represents a caption of a form field. The label can be associated with a specific form control by using the for attribute or by putting the form control inside the label element itself.

Example
Checkbox
Radio 
<div class="c-form__item">
  <label class="c-form__label " for="labelInputText">A label for a text input <span class="c-form__required" aria-label="Required" title="Required">*</span></label>
  <input class="c-form__text" type="text" id="labelInputText" placeholder="Enter some text">
</div>

<fieldset class="c-form__checkboxes">

Section 6.9 forms.select Select list

The <select> element represents a form field for selecting amongst a set of options.

Known limitation: by default, Chrome and Safari on OS X allow very limited styling of <select>, unless a border property is set.

Examples
Default styling
:disabled
Disabled
